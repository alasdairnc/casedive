# casedive — Dive Deep into Canadian Criminal Code

**An educational tool for Justice Studies students and legal professionals.**

🌐 Live: [https://casedive.ca](https://casedive.ca)

> ⚠️ **Educational Tool Only.** This is not legal advice. Always consult a qualified lawyer for legal matters. Verify all citations with official sources like [CanLII](https://www.canlii.org).

---

## What It Does

Describe any criminal scenario in plain language. casedive returns:

- Likely Criminal Code charges with section numbers and severity
- Relevant Canadian case law citations
- Sentencing ranges and related offences
- AI legal analysis of the scenario
- Background citation verification via CanLII

---

## Features

- **Charge identification** — Criminal Code sections, offence titles, severity (summary/indictable/hybrid)
- **Case law** — Relevant Canadian cases with court, year, relevance, and outcome
- **Sentencing panels** — Typical ranges, minimums, maximums per charge
- **Related charges** — Simpler, harsher, and variant offences
- **Search history** — Last 20 queries saved locally with re-run support
- **Filters** — Jurisdiction, court level, date range
- **Dark mode** — Full light/dark theme support

---

## Tech Stack

- **Frontend:** React 18, Vite, inline styles (no CSS framework)
- **API:** Vercel serverless functions
- **AI:** Anthropic Claude (claude-sonnet-4)
- **Citations:** CanLII API (optional)
- **Hosting:** Vercel

## Getting Started

### Local Development

```bash
git clone https://github.com/alasdairnc/casefinder.git
cd casefinder
npm install
cp .env.example .env
# Add your ANTHROPIC_API_KEY to .env
npm run dev
```

### Deploy to Vercel

```bash
npx vercel
npx vercel env add ANTHROPIC_API_KEY
npx vercel --prod
```

## Security

- API key is server-side only — never exposed to the client
- Rate limiting: 10 requests/hour per IP
- Filter values whitelisted server-side (prevents prompt injection)
- Security headers: `X-Frame-Options`, `X-Content-Type-Options`, `Referrer-Policy`
- No user data stored server-side — search history is local-only (localStorage)
